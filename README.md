# Recreation-FNAF

**Five Nights at Freddy's** — реализация классической хоррор-игры в текстовом режиме терминала cmd. Cледите за движением аниматроников и выживайте до 6 утра!

##  Особенности 

-  **Система камер** — наблюдайте за всеми зонами пиццерии
-  **Три уникальных врага** — Чика, Фредди и Бонни с разными маршрутами  
-  **Динамические двери** — закрывайте левую и правую двери офиса
-  test_version **Интеллектуальное ИИ** — враги используют алгоритм Дейкстры для поиска пути
-  test_version **Цветовая индикация** — камеры и зоны отображаются разными цветами


##  Требования

- **C++17** или выше
- **CMake** 3.15+
- **GCC/MinGW** или **MSVC**

##  Сборка и запуск
### for Windows 
 Откройте game.bat файл

## Управление
WASD — движение игрока
Q — выход из игры
Подойдите к красным кнопкам у дверей, чтобы закрыть их
Открытые двери позволяют врагам проникнуть в офис!

## Поведение врагов

Каждый враг имеет свой маршрут 

Чика: SHOW_STAGE → DINING_AREA → KITCHEN → EAST_HALL → OFFICE
Фредди: SHOW_STAGE → DINING_AREA → RESTROOMS → EAST_HALL → OFFICE
Бонни: SHOW_STAGE → DINING_AREA → BACKSTAGE →  SUPPLY_CLOSET → WEST_HALL → OFFICE

## Технические детали

Алгоритм Дейкстры для поиска оптимального пути
Система зон с проверкой допустимых областей для каждого врага
Обработка застревания — телепортация через 10 секунд без движения
Динамические маршруты с случайным выбором между вариантами
